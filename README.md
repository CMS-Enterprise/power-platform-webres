# CMS Power Platform Web Resources

This repository contains custom web resources for Microsoft Power Platform applications supporting Centers for Medicare & Medicaid Services (CMS) initiatives.

## Overview

This repository serves as a centralized location for managing and versioning web resources used across CMS Power Platform solutions, including Power Apps, Power Pages, and Dynamics 365 applications.

## Contents

Web resources in this repository may include:

- **JavaScript Libraries**: Custom scripts for form logic, business rules, and client-side functionality
- **CSS Stylesheets**: Custom styling for model-driven apps and portals
- **HTML Files**: Custom pages, templates, and UI components
- **Images**: Icons, logos, and graphical assets
- **XML Files**: Ribbon customizations and configuration files
- **TypeScript**: Strongly-typed scripts for enhanced development
